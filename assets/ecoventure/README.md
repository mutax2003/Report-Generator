# Ecoventure brand assets

Used by the Streamlit UI (`ui/branding.py`). Sourced from the public site [ecoventure.ca](https://www.ecoventure.ca).

| File | Source |
|------|--------|
| `icon_white.png` | Site header logo (white on slate) |
| `favicon.ico` | Browser favicon |

**Colors** (from site HTML/CSS):

- Slate: `#2e3540`
- Accent magenta: `#b24292`

Refresh assets after a site rebrand:

```powershell
python scripts\fetch_ecoventure_assets.py
```
