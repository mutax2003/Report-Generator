Tag the merge document (when present):

  python scripts/tag_production_template.py --source "22xxxxR Phase 2 ESA Full_merge.docx" --out samples/production_template.docx

Or generate the reference template only:

  python scripts/tag_production_template.py

Run inventory_template.py to list tags after you edit the Word file:

  python scripts/inventory_template.py "samples/production_template.docx"

Bracket placeholders found in the document
------------------------------------------
Replace these in Word with Jinja2 tags (same name, snake_case):

  [Company]           ->  {{ company }}
  [Company Address]   ->  {{ company_address }}
  [Keywords]          ->  {{ keywords }}
  [LAB]               ->  {{ lab_name }}

Common static text to replace
-----------------------------
  "Client Full Name"     ->  {{ client_full_name }}  or  {{ client_name }}
  Site / project ID line   ->  {{ site_name }}
  Address lines            ->  {{ site_address }}  or  {{ address }}
